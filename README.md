Kristal Quest
=============

> Marooned! You must collect crystals to power your spaceship while avoiding hostile life forms.

A Phaser 3 demo made with [brunch-phaser](https://github.com/samme/brunch-phaser).

Controls
--------

- cursor keys
- `D` toggle physics debug
- `Q` quit (to menu)

Features
--------

- advanced slime AI
- arcade physics
- bitmap text
- fruit
- particles
- scene switching
- timer events
- tweens
