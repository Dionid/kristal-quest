Kristal Quest
=============

> Marooned! You must collect crystals to power your spaceship while avoiding hostile life forms.

A Phaser 3 demo made with [brunch-phaser](https://github.com/samme/brunch-phaser).

Controls
--------

- cursor keys
- `D` toggle physics debug
- `F` start full screen (if available)
- `Q` quit (to menu)

Features
--------

- 99 levels
- advanced slime AI
- arcade physics
- bitmap text
- fruit
- game data registry
- grid alignment
- load progress
- particles
- scene data passing
- scene switching
- tiling sprite
- timer events
- tweens
